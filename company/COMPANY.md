ostack is a workflow skill suite for AI-assisted software development. It provides
structured agent roles — reviewer, QA lead, release engineer, debugger, and more —
as composable SKILL.md files that give AI agents specialist capabilities across the
full development lifecycle: planning, building, reviewing, testing, shipping, and
monitoring.
